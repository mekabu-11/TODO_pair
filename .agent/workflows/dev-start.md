# 開発サーバーの起動

このワークフローは、フロントエンドとバックエンドの開発サーバーを起動します。

## 手順

// turbo
1. フロントエンド開発サーバーを起動
```bash
cd frontend && npm run dev
```

// turbo
2. バックエンド開発サーバーを起動（別のターミナルで）
```bash
cd backend && npm run dev
```

## 確認

- フロントエンド: http://localhost:5173
- バックエンド: http://localhost:3000
